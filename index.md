# （仮サイト）滋賀大学競技プログラミングサークル

## テスト

ダミーテキストの代わりに、サイトをこのような形式にした理由を書いておきます。

* いい感じのモダンなデザインにするよりは、このようなシンプルなMarkdown形式で書けるほうが誰もが編集しやすい
* GitHub Pagesで公開することにより、GitHubリポジトリを使える
  * Scrapboxやブログサービスなども編集しやすく良いように思えるが、誰がその編集を行ったのかが明示されない
  * リモートリポジトリから直接編集できる、またその際にプレビューも表示できる
* 2026年度初めの現時点では書くべき内容が比較的少ない（簡単なサークル紹介、過去の大会実績、Twitterアカウントへのリンク、Discordの新歓用サーバーへの招待リンク、程度ならば段落節で分ければこの1ページに収まる）
* 何年か経過した後にサイトを改装する際にためらいがない（フロントエンドの知識が豊富なサークルメンバーが現れた場合など）
